# Images Folder Structure

- `/products/` - Product images (interior.jpg, solid.jpg, divide.jpg, etc.)
- `/team/` - Team member photos
- `/blog/` - Blog post featured images
- `/partners/` - Partner/client logos
- `/materials/` - Material texture images (textiles, denim, etc.)
- `/icons/` - Custom icons and SVGs
- `/og/` - Open Graph images for social sharing (1200x630px)
